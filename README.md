# Pottery-Hub-UXUI-Prototype

Pottery Hub is a **UX/UI prototype** of a pottery-focused website. The goal of this project was to create a **visually appealing and user-friendly interface**, following UX/UI principles, focusing on layout, design consistency, and interactive elements without backend functionality. 

## Features
- Browse pottery collections by category
- Detailed product pages with descriptions and images
- Responsive design for desktop, tablet, and mobile
- Easy navigation and user-friendly interface

## Tools Used
- **HTML, CSS, JavaScript** – for building the static prototype
- **W3Schools tutorials and references** – to learn and guide the template creation

## What I Did
- Designed a **homepage and other pages** following UX/UI principles
- Created **responsive layouts** for desktop and mobile screens
- Implemented **navigation and interactive elements** using HTML, CSS, and JavaScript
- Focused on **aesthetics, usability, and intuitive user flow**


## Outcome
- A **fully designed UX/UI prototype** of a pottery website
- Visual representation of **user-friendly navigation and layout**
- Demonstrates **responsive design** principles
- Ready to be further developed into a fully functional website in the future
